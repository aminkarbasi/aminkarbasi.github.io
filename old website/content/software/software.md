We highly believe in open and reproducible science. To this end, we try our best to publicly release our software and data sets.

- Paper [Submodular Maximization beyond Non-negativity: Guarantees, Fast Algorithms, and Applications](http://proceedings.mlr.press/v97/harshaw19a/harshaw19a.pdf):
	- Code is released [here](https://github.com/crharshaw/submodular-minus-linear)

- Paper: [Streaming Weak Submodularity: Interpreting Neural Networks on the Fly](https://arxiv.org/pdf/1703.02647.pdf)
	-  Code is released [here](https://github.com/eelenberg/streak).

- Paper: [Probabilistic Submodular Maximization in Sub-Linear Time](http://proceedings.mlr.press/v70/stan17a.html)
	- Code is released [here](https://github.com/serbanstan/TwoStage)

- Paper: [Differentially Submodular Maximization: Data Summarization in Disguise](http://proceedings.mlr.press/v70/mitrovic17a.html)
	- Code is released [here](https://github.com/mmitrovi/submodSeq).

- Paper [Online Continuous Submodular Maximization](http://proceedings.mlr.press/v84/chen18f/chen18f.pdf): 
	- Code is released [here](https://github.com/lchen91/Continuous_Submodular_Maximization)
